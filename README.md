# c-bff

Simple [Brainfuck](https://esolangs.org/wiki/Brainfuck) interpreter written in C.

# Building

```
$ make bin
```

# Running

```
$ ./bin/c-bff tests/hello.bf
Hello, world!
```

# Testing

```
$ make test
```

# Limitations

Many. I'm still improving it. Feel free to open a PR if you want to contribute!
